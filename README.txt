Para poder correr el api rest deben instalar el IDE Netbeans y el servidor XAMPP.
1)Ingresamos a xampp y activamos el apartado de Apache y Mysql
2En el navegador ingresamos a la direccion http://localhost/phpmyadmin/
3)Abrimos el query que esta en el archivo bd.txt, lo copiamos y luego lo pegamos en el apartado de query en el gestor de base de datos y lo corremos.
4) Abrimos el proyecto desde netbeans, le damos click derecho y luego clickeamos la opcion "Test Restful services"
5) Desplegara el navegador y desde ahi podemos realizar las peticiones, ya nos otorga la url de las mismas
